# 💻 Fundamentos em C

## 🚀 Projetos

* ⏱️ **timed_readline**
  Implementação própria de leitura de entrada com tempo limite usando a system call `select()` do Linux.

* 📦 **Input_Library**
  Biblioteca para entrada de dados em C, incluindo suporte a strings de tamanho desconhecido com gerenciamento automático de memória.

* 🧩 **data_structures**
  Implementações simples de estruturas de dados fundamentais.

* 🔀 **merge_sort**
  Implementação do algoritmo de ordenação merge sort para arrays numéricos.

* 🔐 **criptografia_ceasar**
  Programa que aplica cifra de César com chave definida pelo usuário.

* 🧮 **calculadora_areas**
  Cálculo de áreas de polígonos utilizando function pointers.

* 🏦 **Cartorio**
  CRUD simples em C (Windows).
